方法二：v-for 可以通过一个对象的属性来迭代数据，指令：value in object

  • {{value}}

方法二点一：可以提供第二个的参数为键名：(value, key) in object

  • {{ key }} : {{ value }}

方法二点二：第三个参数为索引：(value, key, index) in object

  • {{ index }}. {{ key }} : {{ value }}
